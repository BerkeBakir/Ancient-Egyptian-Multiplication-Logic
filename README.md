 # Antik Mısır Çarpım Algoritması  Bu C programı, iki sayıyı Antik Mısır'da kullanılan yönteme benzer bir algoritma ile çarpar. Algoritma, çarpma işlemini temel olarak toplama, ikiye bölme ve ikiyle çarpma gibi daha basit operasyonlara indirger.  ## Algoritma Nasıl Çalışır?  Antik Mısır çarpım algoritması, bir sayıyı sürekli olarak ikiye bölme ve diğer sayıyı sürekli olarak ikiyle çarpma prensibine dayanır. İlk sayının (num1) çift veya tek olduğu kontrol edilir. Eğer sayı tek ise, ikinci sayı (num2) sonuca eklenir. Bu işlemler, ilk sayı sıfıra ulaşana kadar devam eder.  ### İşlem Adımları  1.  İlk sayı (`num1`) ikiye bölünür (tam sayı bölmesi). 2.  İkinci sayı (`num2`) ikiyle çarpılır. 3.  Eğer ilk sayı (`num1`) tek ise, ikinci sayının güncel değeri toplam sonuca eklenir. 4.  Bu adımlar, `num1` sıfır olana kadar tekrarlanır. 5.  Sonuç olarak, tüm eklenen sayıların toplamı, iki sayının çarpımını verir.  ## Kullanım  ### Derleme  Projeyi derlemek için bir C derleyicisine (örneğin GCC) ihtiyacınız vardır.
